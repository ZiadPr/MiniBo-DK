# MiniBo Systems

تطبيق تشغيل فعلي لإدارة إدخال الإنتاج والاعتماد والمتابعة الإدارية.

## التشغيل

```bash
npm install
npm run dev
```

ثم افتح:

`http://localhost:3000`

## بيانات الدخول

- محلياً فقط وبشكل افتراضي: `admin / 1234`
- محلياً فقط وبشكل افتراضي: `supervisor / 1234`
- في بيئة الإنتاج اضبط `MINIBO_ADMIN_PASSWORD` و`MINIBO_SUPERVISOR_PASSWORD`
- في `production` يتم إيقاف الاعتماد على الحسابات التجريبية افتراضياً ما لم تعيّن `MINIBO_ALLOW_DEMO_LOGIN=true`

## التخزين

- عند وجود `DATABASE_URL` سيستخدم التطبيق PostgreSQL تلقائياً
- عند عدم وجود `DATABASE_URL` سيستخدم الملف `data/minibo-store.json`
- الملف المحلي يُنشأ تلقائياً عند أول تشغيل
- أي تعديل في جلسات الإنتاج أو المنتجات يُخزن بشكل دائم في الـ backend المختار

## PostgreSQL

- أضف `DATABASE_URL` في ملف البيئة
- يمكن استخدام ملف [database/postgres/init.sql](</h:/Minibo vite/database/postgres/init.sql>) إذا أردت تجهيز الجدول يدوياً
- إذا لم تُجهز الجدول يدوياً فالتطبيق سيُنشئه تلقائياً عند أول اتصال
- التخزين في PostgreSQL أصبح relational schema فعلياً وليس JSONB واحداً
- إذا وُجد تخزين قديم في جدول `minibo_app_store` فسيتم استيراده تلقائياً عند أول تشغيل على القاعدة الجديدة

## التشخيص

- صفحة `/admin` تعرض بطاقة "حالة التخزين" لمعرفة الـ backend الحالي
- يوجد endpoint مباشر للفحص على `/api/system/storage-status`
- البطاقة تعرض حالة الاتصال، جاهزية الـ schema، وعدد السجلات الحالية

## الخطوط العربية

- الواجهة تستخدم `Cairo` بشكل مباشر على `body` وعناصر MUI
- تصدير `PDF` يستخدم خطًا عربيًا محليًا مضمّنًا افتراضيًا: `public/fonts/Amiri-Regular.ttf`
- يمكن override عبر `PDF_ARABIC_FONT_PATH` إذا أردت خطًا مختلفًا على الخادم

## ما هو جاهز الآن

- تسجيل دخول وحماية مسارات
- لوحة تحكم تشغيلية
- شاشة إدخال إنتاج مع حفظ دائم
- ملاحظات إنتاج مرتبطة بالجلسة
- اعتماد وطباعة
- تصدير Excel وPDF من صفحة الاعتماد
- متابعة الإنتاجية المطلوبة مقابل الفعلي
- إدارة منتجات فعلية من لوحة الإدارة
- تشخيص مباشر لحالة التخزين
- smoke test جاهز للتحقق من التشغيل بعد النشر

## ملاحظة

إذا أردت تشغيله في بيئة إنتاجية خارجية، عيّن `JWT_SECRET` ويفضل `DATABASE_URL` في ملف البيئة بدلاً من الاعتماد على القيم الافتراضية.

يمكنك أيضاً تعيين `PDF_ARABIC_FONT_PATH` إذا أردت إجبار مولد PDF على استخدام خط عربي محدد بدلاً من الاعتماد على الخطوط الافتراضية الموجودة في النظام.
